# Changelog

All notable changes to this project will be documented in this file.
## v0.12 – 2026-03-16

### Added
- Undo functionality to revert the last drawing action
- Clear canvas button to reset the drawing area

### Improved
- Eraser tool now displays the erasing range for better user feedback


## v0.11 – 2026-03

Added:

- Brush thickness control
- UI panel for drawing settings

Improved:
- Smoother mouse drawing performance
- Better canvas pan navigation
